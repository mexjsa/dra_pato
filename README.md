# Dra. Ana Patricia Arellano - Clínica Dental

Landing page premium, moderna y responsive para la clínica dental de la Dra. Ana Patricia Arellano ubicada en San Pedro Cholul, Mérida, Yucatán.

## Características
- **Alineado a Diseño Premium**: Diseño basado en HSL con degradados color menta/turquesa y componentes glassmorfistas.
- **Formulario Inteligente**: Formulario de reserva de cita integrado directamente con la API de WhatsApp, direccionando la información al número **9991402191**.
- **Mapa Interactivo**: Ubicación geográfica interactiva mediante Google Maps embebido para encontrar la dirección fácilmente.
- **SEO Optimizado**: Etiquetas meta estructuradas para mejorar el posicionamiento local.
- **Responsividad Total**: Optimizado tanto para dispositivos móviles como pantallas de escritorio grandes.

## Estructura del Proyecto
- `index.html`: Estructura semántica HTML5 de la página.
- `styles.css`: Estilos premium, variables personalizadas, tipografía y responsive design.
- `script.js`: Comportamiento interactivo de navegación y validación/redirección de WhatsApp.
- `assets/hero_dental.png`: Imagen del Hero generada profesionalmente.
